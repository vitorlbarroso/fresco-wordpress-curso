Site desenvolvido em WordPress, seguindo o curso do professor Guanabara (Curso em Vídeo). No curso é apresentado o WordPress e nele criamos um site para um pizzara fictícia, chamada Fresco. O projeto consta com uma página inicial de recepção, área do cardápio, apresentação do restaurante, aba de notícias e contato. Utilizamos nesse projeto o Tema Astra e o Editor de Blocos Gutenberg Pro.
